The site you're currently viewing. A personal portfolio and blog built with modern web tooling.

## Features

- **MDX content** — blog posts and project pages written in Markdown with embedded React components, processed by Contentlayer
- **Dark theme** — custom CSS with a slate/indigo colour scheme and gradient accents
- **Responsive design** — works across desktop and mobile
- **Blog cards** — auto-generated grid with cover images, dates, and summaries
- **CV viewer** — embedded PDF viewer with download link
- **Deployed on Vercel** — automatic redeployment on push to main

## Tech Stack

Next.js, React, TypeScript, MDX, Contentlayer, Vercel
